PROJECT INTRODUCTION

        
Folder encryption describes the process of encrypting folders, including files, which means that no one has access and can read the data unless they have a specific decryption key. 
            The purpose is to prevent malicious or unauthorized parties from accessing sensitive data. 
            Folder encryption is a security measure that involves the use of encryption techniques to protect the contents of a specific folder or directory on a computer or storage device. This process ensures that only authorized users with the correct decryption key or password can access the data stored within the encrypted folder.
            There are different types of encryption techniques like symmetric encryption, asymmetric encryption, password-based encryption. Symmetric encryption uses the same key for both encryption and decryption. The challenge is securely sharing the key among authorized users. Asymmetric encryption utilizes a pair of public and private keys. Data encrypted with the public key can only be decrypted with the corresponding private key. Password-based encryption uses a password or passphrase to derive an encryption key. Users need to enter the correct password to access the encrypted folder.
